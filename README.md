# Java Tutorial - Basic Concepts

A comprehensive collection of Java programs covering fundamental programming concepts and language features for beginners.

## 📚 Project Overview

This repository contains Java source files that demonstrate core Java programming concepts. Each file focuses on a specific topic with practical examples and explanations.

## 🚀 Getting Started

### Prerequisites
- Java Development Kit (JDK) 8 or higher
- A code editor or IDE (VS Code, IntelliJ IDEA, Eclipse, etc.)

### How to Run

1. **Compile a Java file:**
   ```bash
   javac FileName.java
   ```

2. **Run the compiled program:**
   ```bash
   java FileName
   ```

**Example:**
```bash
javac Hello.java
java Hello
```

## 💡 Learning Path

**Recommended order for beginners:**

1. Start with `Hello.java` - Get familiar with basic syntax
2. Learn `Arithematic.java` - Understand operators
3. Explore `Literal.java` and `Conversion.java` - Data types
4. Move to conditional statements: `Conditional_Statements.java` → `Else_if.java` → `Switch.java` → `Ternary.java`
5. Master loops: `For_Loop.java` → `While_loop.java` → `Do-While.java`
6. Understand operators: `Relational.java` → `Logical.java`
